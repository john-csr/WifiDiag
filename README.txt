

# Wi-Fi Direct + Miracast Diagnostic Tool

## Overview

**Wi-Fi Direct + Miracast Diagnostic Tool** is a lightweight HTA-based Windows application that checks system readiness for Wi-Fi Direct and Miracast. It also provides manual control over the WLAN AutoConfig service, making it ideal for AV/IT professionals troubleshooting wireless display capabilities.

This tool runs silently using embedded PowerShell and presents results in a clean, dark-themed interface.

---

## Features

- **Wi-Fi Direct and Miracast Diagnostics**: Run full system checks for wireless display support  
- **WLAN AutoConfig Control**: Start, stop, or restart the WLAN service manually  
- **Service Status Check**: View current WLAN service state  
- **Live Output Display**: See PowerShell results in real time  
- **Minimal Interface**: HTA-based GUI with no external dependencies  

---

## Requirements

- Windows 10 or later  
- Administrator privileges  
- HTA support (enabled by default on most Windows systems)  

---

## How to Use

1. Download and save `WiFiDiag.hta` to your desktop  
2. Double-click the file to launch the tool  
3. Use the interface buttons to run diagnostics or manage the WLAN service  
4. View results in the live output box
