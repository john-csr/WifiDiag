# Wi-Fi Direct + Miracast Diagnostic Tool

A lightweight Windows HTA application that checks system readiness for Wi-Fi Direct and Miracast, and provides manual control over the WLAN AutoConfig service.

## Features

- Run full diagnostics for Wi-Fi Direct and Miracast
- Restart the WLAN AutoConfig service manually
- Check WLAN service status
- Clean, professional interface with dark theme
- Silent PowerShell execution with live output display

## Requirements

- Windows 10 or later
- Administrator privileges
- HTA support (enabled by default on most Windows systems)

## How to Use

1. Download and save `WiFiDiag.hta` to your desktop
2. Double-click to launch the tool
3. Use the buttons to run diagnostics or manage the WLAN service
4. View results in the live output box
